Sistema de Elecciones

El proyecto Sistema de Elecciones se concibe en Lacnic en el año 2018. Existía una necesidad de sistematizar las elecciones de cargos internos de forma de tener elecciones transparentes, fiables y manteniento el voto secreto. Para asegurar la transparencia del proceso existen auditores que validan y verifican una elección. Debido a la naturaleza distribuida de los usuarios (votantes, administradores, auditores), se decidió que el sistema fuese una aplicación web el cual envía los enlaces de votación por correo con un identificador único. Cada identificador puede ser utilizado una única vez, pero el sistema permite votar por uno o varios cargos, dependendiendo la de configuración de la elección.

Inicialmente el uso fue exlusivo de Lacnic para la elección de cargos de su comunidad. Luego, al cabo de un tiempo, otras organizaciones aliadas, con necesidades similares, al tomar conocimiento del sistema lo comenzaron a solicitar para sus propias eleciones. En este momento fue que surgió la idea de aperturar el código y disponibilizarlo para que otras comunidades puedan utilizar el sistema y beneficiarse mediante elecciones transparentes y fiables.

En las siguientes secciones se puede encontrar información sobre el repositorio del código fuente, documentación para la instalación y configuracion. También documentos y manuales que permitan guiar al usuario para la creación y administración de elecciones, así también como realizar el proceso de votación, el proceso de auditar.
